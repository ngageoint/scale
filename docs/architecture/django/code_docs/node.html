class node.models.Node(*args, **kwargs)¶

Bases: django.db.models.base.Model

Represents a cluster node on which jobs can be run

Parameters:
  • hostname (django.db.models.CharField) – The full domain-qualified hostname of the node
  • port (django.db.models.IntegerField) – The port being used by the executor on this node
  • slave_id (django.db.models.CharField) – The slave ID used by Mesos for the node
  • pause_reason (django.db.models.CharField) – User or system specified reason why this node is paused. Should be used for display only.
  • is_paused (django.db.models.BooleanField()) – True if the node is currently paused and should not accept new jobs
  • is_paused_errors (django.db.models.BooleanField()) – If the node is paused, was it do to a high error rate?
  • is_active (django.db.models.BooleanField()) – True if the node is currently active or is archived for historical purposes
  • created (django.db.models.DateTimeField) – When the node model was created
  • archived (django.db.models.DateTimeField) – When the node was archived/deactivated
  • last_offer (django.db.models.DateTimeField) – When the node last received an offer from mesos (regardless of if the offer was used)
  • last_modified (django.db.models.DateTimeField) – When the node model was last modified
